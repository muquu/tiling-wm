setup
=========

```
$ git clone git:://github.com/muquu/tiling-wm ~/path/to
$ mkdir -p ~/.config/awesome
$ ln -s ~/path/to/awesome/rc.lua ~/.config/awesome/rc.lua
```

or

```
$ git clone git:://github.com/muquu/tiling-wm ~/path/to
$ mkdir -p ~/.xmonad
$ ln -s ~/path/to/xmonad/xmonad.hs ~/.xmonad/xmonad.hs
```
